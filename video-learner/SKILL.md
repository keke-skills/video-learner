# Video-Learner 🎥

把视频变成可用的 Skill！

## ⚠️ 重要：两种使用方式

### 方式1：直接发链接给我（推荐！）

**最简单！不需要任何配置！**

- 直接把抖音/B站/YouTube 链接发给我
- 我自动：下载 → 识别 → 分析 → 生成 Skill → 保存
- 我会用我的 LLM 能力自动处理，不需要你配置任何东西

### 方式2：手动运行脚本

需要安装依赖，然后运行脚本。

---

## 支持平台

- 🎵 **抖音** - 无水印视频下载 + 语音识别
- 📺 **B站** - 音频提取 + 语音识别  
- 🎬 **YouTube** - 音频提取 + 语音识别

## 功能

- **视频下载** - 支持抖音/B站/YouTube视频/音频提取
- **语音识别** - 用Whisper把音频转成文字
- **LLM分析** - 提取核心知识点、适用人群、难度等级
- **用户确认** - 分析结果展示给你，确认后再生成完整 Skill
- **自动安装** - 生成的 Skill 自动安装到 skills/ 目录
- **可调用** - 生成后可直接使用

## 使用方法

### 方式1：直接发给我（推荐！）

```
把视频链接发给我 → 我自动生成 Skill！
```

### 方式2：手动运行脚本

```bash
# 安装依赖
pip3 install yt-dlp openai-whisper
brew install ffmpeg node

# 运行
python3 video-learner.py "视频链接"
```
